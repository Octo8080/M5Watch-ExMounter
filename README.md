# M5Watch-ExMounter
These files on the repository are M5Watch extensions module.

## Notes
If you use it, you need a 3D printer or an alternative.  
Add thereas needed.  
- 3 mm screws  
- 2 mm  springs   
- 3 mm metal pipe   

Use only the watchband from the original M5watch when assembling.
